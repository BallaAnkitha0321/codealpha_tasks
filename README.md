## Objective
Develop a machine learning classification model to accurately identify Iris flower species based on their sepal and petal measurements.

## Dataset Features
- Sepal Length (cm)
- Sepal Width (cm)
- Petal Length (cm)
- Petal Width (cm)

## Target Variable
- Iris Setosa
- Iris Versicolor
- Iris Virginica

## Machine Learning Technique
- Classification
- Random Forest Classifier

## Analysis Performed
- Data Exploration
- Feature Visualization
- Train-Test Split
- Model Training
- Prediction
- Performance Evaluation

## Results
- High classification accuracy on the test dataset.
- Successfully classified all three Iris species.
- Generated predictions for user-provided flower measurements.

## Outcome
Built a robust classification model capable of accurately predicting Iris flower species from input measurements.

# Task 3: Car Price Prediction using Machine Learning

## Objective
Develop a regression model to estimate the selling price of used cars based on vehicle characteristics.

## Features Used
- Present Price
- Driven Kilometers
- Fuel Type
- Selling Type
- Transmission
- Owner
- Car Age (Feature Engineered)

## Target Variable
- Selling Price

## Machine Learning Technique
- Regression
- Random Forest Regressor

## Model Performance
- Mean Absolute Error (MAE): **≈ 0.88**
- R² Score: **≈ 0.96**

## Feature Importance

| Feature | Importance |
|----------|-----------:|
| Present Price | 0.9091 |
| Car Age | 0.0682 |
| Driven Kilometers | 0.0142 |
| Transmission | 0.0048 |
| Selling Type | 0.0020 |
| Fuel Type | 0.0017 |
| Owner | 0.0000 |

## Key Findings
- Present Price had the highest influence on predicting selling price.
- Car Age was the second most important feature.
- Driven Kilometers had a moderate impact.
- Owner information contributed very little to the prediction.

## Outcome
Built an accurate machine learning model capable of estimating used car prices with excellent prediction performance.

# Task 4: Sales Prediction using Python

## Objective
Predict product sales using advertising expenditure across TV, Radio, and Newspaper marketing channels.

## Features Used
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget

## Target Variable
- Sales

## Machine Learning Technique
- Regression
- Linear Regression

## Model Performance
- Mean Absolute Error (MAE): **≈ 1.25**
- R² Score: **≈ 0.90**

## Key Findings
- TV advertising had the strongest positive impact on sales.
- Radio advertising contributed moderately to sales prediction.
- Newspaper advertising showed the least influence.
- The model successfully predicts future sales based on advertising investment.

## Outcome
Developed a regression model capable of estimating expected product sales using advertising budgets from multiple marketing channels.
